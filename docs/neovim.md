# Neovim

Vim text editor. Download [here](https://neovim.io/)

![my-environment](./images/noevim.png)


## Dependencies

- **Universal-ctags**
```
brew install --HEAD universal-ctags/universal-ctags/universal-ctags
```

or 

```
sudo apt-get install exuberant-ctags
```
